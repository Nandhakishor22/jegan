1.Html tag

The <html> is used to represent the root of an html Document
The <html> tag is the container for all other hyml elements (except for the <!Doctype> tag)
While using this html tag the developer always include the lang attribute inside the html tag, to declare the of the webpage. This is meant to assist search engines and browsers.
ex:
<!DOCTYPE html>
< html lang="en" >
 <head>
 </head>
 <body>
 </body>
</html>

2.Head tag

The <head> tag is used to define the head portion of the document which contains the information about the document.
The <head> contains other head elements such as

<title>
<meta>
<link>
<style>

The Metadata is data about the html document. Metadata is not displayed. Metadata typically define the document titile, character set, styles, scripts, and other meta information. ex:
<!DOCTYPE html>
<html>
 <head>
  <meta charset="utf-8">
  <title>My test page</title>
 </head>
 <body>
  <p>This is my page</p>
 </body>
</html>

3.Body tag

HTML <body> tag defines the main content of an HTML document which displays on the browser. It can contain text content, paragraphs, headings, images, tables, links, videos, etc.
The <body> must be the second element after the <head> tag or it should be placed between </head> and </html> tags. This tag is required for every HTML document and should only use once in the whole HTML document.
syntax:
<body>Content goes here .......</body>
ex:
<!DOCTYPE html>
<html>
 <head>
 </head>
 <body>
  <h2>Example of body tag</h2>
  <p>This line is written inside the body tag</p>
 </body>
</html>

4.Title tag

The <title> tag defines the title of the document The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
It sets the title in the browser toolbar. It provides the title for the web page when it is added to favorites. It displays the title for the page in search engine results.
The <title> element
 1.defines a title in the browser toolbar
 2.provides a title for the page when it is added to favorites
 3.displays a title for the page in search-engine results
A web page canno have more than one <title> tag
syntax:
<title> Title name </title>
ex:
<!DOCTYPE html>
<html>
 <head>
  <title> Title name </title>
 </head>
 <body>
  <h2>Body</h2>
 </body>
</html>

5.H1 tag

HTML headings are defined with the <h1> to <h6> tags A HTML heading or HTML h tag can be defined as a title or a subtitle which you want to display on the webpage. When you place the text within the heading tags <h1>.........</h1>, it is displayed on the browser in the bold format and size of the text depends on the number of heading.
<h1> defines the most important heading and <h6> defines the least important heading
Search engines use the headings to index the structure and content of your web pages.
Ex:
<h1>Heading no. 1</h1>
<h2>Heading no. 2</h2>
<h3>Heading no. 3</h3>
<h4>Heading no. 4</h4>
<h5>Heading no. 5</h5>
<h6>Heading no. 6</h6>

5.P tag

The <p> tag defines a paragraph.
Browsers automatically add asingle blank line before and after each <p>. An html <p> indicates the starting of the paragraph.
The browser ignores the lines in the paragraph in the source code to make the line break the user need to use the <br> tag.
Ex:
<!DOCTYPE html>
<html>
 <head>
 </head>
 <body>
  <p>This is a paragraph</p>
  <p>This is a paragraph</p>
  <p>This is a paragraph</p>
 </body>
</html>
